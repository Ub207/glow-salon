# Salon AI Bot

**Live Demo:** https://salon-ai-bot.vercel.app

An AI-powered chatbot and booking system for salons. Built with Node.js, Express, and Groq AI (LLaMA 3.3 70B).

## Features

- AI chatbot assistant for salon inquiries
- Appointment booking form
- Fast responses powered by Groq API

## Tech Stack

- **Backend:** Node.js + Express
- **AI:** Groq API (llama-3.3-70b-versatile)
- **Deployment:** Vercel

## Setup

1. Clone the repo
2. Create a `.env` file with your `GROQ_API_KEY`
3. Run `npm install` then `npm start`
